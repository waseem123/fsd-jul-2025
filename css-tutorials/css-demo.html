Cascading Stylesheet(CSS)

Cascading stylesheet is a document used for improving the look and feel of the html page structure.

Syntax to work with css - 

         css_property:value

Types of CSS

CSS is divided in three categories -

  1. Inline CSS - Applied within inside the tag body. syntax - 
                style = "css_property1:value;css_property2:value;"
  2. Internal CSS
  3. External CSS
