Functions in JS

Function is a self contained block of code that executes a given task.

function has 4 types

  1. No parameter no return
  2. Parameter No return
  3. No parameter with return
  4. Parameter and return

Syntax to declare a function

        function x;

Syntax to define a function

        function nameOfFunction(p1,p2...pN){
            statement 1
            statement 2
            ...
            statement N
        }

Syntax to call a function

        nameOfFunction(arg1,arg2...argN);
